# Developer
## To Generate g package
`flutter packages pub run build_runner build`

## How to release app
- Change the version at pubspec.yaml ->  eg. `version: 1.1.0+2`. `1.1.0` for version name and `2` for version code
- Run `flutter build appbundle`
